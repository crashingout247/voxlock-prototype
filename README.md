# VoxLock: AI Voice Isolation Web App

Frontend (JS/MediaPipe): Adeline
Backend (Flask): Cherry

## Quick Start
- Clone: `git clone https://github.com/crashing02/voxlock-prototype.git`
- Frontend: Open `frontend/index.html` with Live Server (port 3000).
- Backend: `cd backend && pip install -r requirements.txt && python app.py` (port 5000).

For collab: Pull requests welcome!
